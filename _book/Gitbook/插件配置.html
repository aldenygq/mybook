Gitbook配置

一般的，需要在根目录中新增一个文件名为：book.json，来作为整本书籍的配置文件

$ vim book.json
{
    "title": "古道长亭",  //标题
    "author": "古道长亭", //作者
    "description": "welcome to my docs", //书描述

    "plugins": [    //插件列表

插件

Gitbook默认自带有5个插件:

  • highlight： 代码高亮
  • search： 导航栏查询功能（不支持中文）
  • sharing：右上角分享功能
  • fontsettings：字体设置（最上方的"A"符号）
  • livereload：为GitBook实时重新加载

    如果不用可以前面加-去掉

安装插件

$ gitbook install

插件配置可查看GitBook插件整理
